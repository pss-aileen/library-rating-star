## How to use it?

### 1. スクリプト

#### シンプル

```javascript
new RaitingStar();
```

#### オプション付き

```javascript

  filledColor: 'pink', // 色
  emptyColor: 'red', // 色
});
```

### 2. HTML

```html

<div class="rs">2.5/5</div>
```
